N-Body Simulation

N-Body Simulator is a 2D N-body simulation written using the p5 javascript library. It features accurate N-Body gravity simulation, a randomly generated starry backdrop, and a rudimentary circles-only physics engine. (no rotation)

The Factory

The Factory is short, story-driven platformer, fully developed by myself. Made in Construct 3, the game has a pixel-art style and retro-ish music made with beepbox. The gameplay is a mix of a metroidvania, precision platformer, and a Super-Smash-Bros-Esque Beat-em-up.

Try to Leave

Try to Leave is another short, story-driven platformer made in Construct 3. Based on Jordan Peele's Get Out, Try to Leave attempts to instill the same message through vastly different media.

Blocky Shader

Blocky is a vanilla inspired shader for Minecraft Java, writen in GLSL. Its features include Shadows, Fog, PBR (Normals and Specular), Bloom, FXAA, and a Dynamic Day/Night Lighting.
